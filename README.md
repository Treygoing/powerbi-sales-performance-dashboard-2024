# PowerBI-Sales-Performance-Dashboard-2024
Interactive Power BI dashboard analysing 2024 retail sales performance, revenue trends, and regional contribution.
## Overview
This Power BI project analyses retail sales performance using a 2024 sample dataset.  
The dashboard was built to practise data modelling, DAX measures, and presenting insights in a clean, decision-focused way.

## Problem Statement

Retail managers need visibility into:

- Revenue trends across the year  
- Best and worst performing months  
- Top contributing states  
- Whether growth is driven by pricing or volume  

This dashboard was built to answer these questions in an interactive format.

---
## Dataset

- Sample retail sales dataset  
- Transaction-level sales data  
- Fields include:
  - Order Date
  - State
  - Product
  - Units Sold
  - Revenue  

Data was cleaned and structured for consistent time-based analysis.

---
## What I Did

- Cleaned and transformed raw sales data  
- Built a proper date hierarchy for monthly trend analysis  
- Created DAX measures for:
  - Total Revenue  
  - Total Units Sold  
  - Average Price per Unit  
- Designed KPI cards for high-level performance tracking  
- Built interactive slicers for product-level filtering  
- Structured the layout for clarity and business storytelling  

---

## Key Insights

- **$6.28M** total annual revenue  
- **12.47M** total units sold  
- **$0.48** average price per unit  
- Stronger performance mid-year and toward year-end  
- Revenue concentrated in key states:
  - California  
  - New York  
  - Florida  
- Business model reflects a high-volume, low-margin structure  

---

## Business Value

This dashboard helps decision-makers:

- Identify seasonal trends  
- Detect high-performing regions  
- Monitor pricing vs volume contribution  
- Focus strategy on top revenue drivers
